# RULES — Olive & Vine 한글 카피 다듬기 규칙 (최신본)

> 작업 본질: 영→한 신규 번역이 아니라 **기존 한국어 문장의 품질 개선**. 코드 구조는 절대 보존, 한국어 string 값만 교체.

## 핵심 원칙
1. **정확성** — 원래 의미·정보를 왜곡·누락하지 않는다. 직역투는 그대로 두지 않는다.
2. **자연스러움** — 한국 B2B 프리미엄 사이트답게 읽히도록. 단, **이미 자연스러운 문장은 무리해서 바꾸지 않는다(과도한 손질 금지).**
3. **일관성** — 용어·문체·톤을 사이트 전체에서 통일.

## UI 요소별 문체
| 유형 | 문체 |
|------|------|
| 헤딩/타이틀 | 핵심 명사 단독, 군더더기 없이 |
| 본문(단락) | 완성형 문어체 종결 (~합니다 / ~해 드립니다 / ~지향합니다) |
| 불렛 | **명사형 종결**, 공백 포함 25자 내외, 병렬 구조. 종결어미(~합니다) 금지. 나열은 가운뎃점(·)으로 |
| 버튼/CTA | 짧은 동사형·명사형 행동 유도 |
| 라벨/메타 | 간결한 명사구 |

## 톤 & 보이스
- 전문적·신뢰감 있는 세련된 문어체. 과장·구호 절제.
- 동행·신뢰 어조: 권위적 단정보다 함께한다는 결. `함께 고민합니다`, `곁에서`, `~이 먼저입니다`.
- 인칭(페이지별 적용 — v6 확정):
  - **서비스 페이지·회사 소개(About) 등 공식 면 → `저희`** (예: corporate "저희는 설립, 운영, 정리…")
  - **가치(Values)·브랜드 서사 면 → `우리` 허용** (예: our-values "우리가 돌아오는 다섯 가지 원칙")
  - **페이지명 성격의 고정 메뉴 라벨(`우리의 이야기/가치/리더십` = Our story/values/leadership)은 `우리의 X`로 통일** (페이지마다 우리/저희로 흔들리지 않게 고정)
  - you/your → 고객/여러분 최소 사용, 맥락 흡수. 남발 금지.
- 고유명사: 브랜드명·인명 영문 표기 유지(음차 금지). 본명(박지현·김미영)은 실명이므로 예외.
- 호흡: 한 문장 한 호흡. 긴 문단은 분할.

## 라벨 위계 규칙 (리더십)
- **섹션 구분 라벨(eyebrow/섹션 헤딩) → `주요 리더십`** (예: About·Leadership 페이지의 창업자 섹션 상단 라벨)
- **개인·카드 단위 역할 라벨 → `공동창업자`** (각 인물 카드 상단)
- 원칙: 큰 묶음(섹션)은 '리더십', 세부(개인)는 구체 직함('공동창업자')으로 위계를 구분한다.

## 번역투(calque) 금지
| 영문 | 금지 | 권장 |
|------|------|------|
| deserve | "받을 자격이 있다" | 펌의 신념·제공 가치로 재구성 |
| more than X | "X 이상의 것" | "X에서 끝나지 않는다 / X를 넘어" |
| their/them | "그들의/그들을" 남발 | 생략·맥락 흡수 |
| sounding board | "울림판" | "함께 고민하는 상대" |
| advocate | "옹호자" | "든든한 지원군" |
| 수동태·명사구 나열 | "~에 의하여" | 능동·서술형 재배치 |
